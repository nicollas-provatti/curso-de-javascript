# 📝 Exercícios

---

### 🔹 Exercício 1: Pode votar

**Descrição:** Crie um programa que leia o nome e a idade de uma pessoa. Se a idade for **maior ou igual** a 16, imprima: ``[nome], você já pode votar!``. Caso contrário, imprima ``[nome], você ainda não pode votar!``. Onde [nome] é o nome da pessoa.

**Exemplo de:**

- Entrada:
  ```css
  Qual o seu nome ? Lucas
  Qual a sua idade ? 17
  ```

- Saída:
  ```css  
  Lucas, você já pode votar!
  ```

---

### 🔹 Exercício 2: Maior de Idade

**Descrição:** Crie um programa que leia o nome e a idade de uma pessoa. Se a idade for **maior ou igual** a 18, imprima ``[nome], você é maior de idade!``. Caso contrário, imprima ``[nome], você é menor de idade!``. Onde [nome] é o nome da pessoa.

**Exemplo de:**

- Entrada:
  ```css
  Qual o seu nome ? Marcos
  Qual a sua idade ? 16
  ```

- Saída:
  ```css  
  Marcos, você é menor de idade!
  ```

---

### 🔹 Exercício 3: Verifica Acesso

**Descrição:** Crie um programa que leia uma senha numérica. Se a senha for **1234**, o programa deve mostrar ``“Acesso permitido”``. Caso contrário, deve mostrar ``“Acesso negado”``.

**Exemplo de:**

- Entrada:
  ```css
  Informe a senha: 1234
  ```

- Saída:
  ```css  
  Acesso permitido.
  ```
